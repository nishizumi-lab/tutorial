# VSCodeで画像編集できるプラグイン「Luna Paint」の使い方

なんと「Luna Paint」のというVScode上で画像編集できるプラグイン(拡張機能)があります。
本格的な画像編集には向いていませんが、字や図形を入れたり、トリミングしたりするのには手軽に使えて向いています。
手軽にメニューから画像のクロップなど出来るのは素敵。

「Luna Paint」のインストール方法と使い方は以下のとおり。

- VScodeを開き、「拡張機能のアイコン」をクリック。

- 「Luna paint」と入力し、「インストール」をクリック。

- 画像ファイルを開きます。

- 例えば、画像のサイズを変更したい場合、「Menu」→「Resize」で倍率をパーセントで入力すると、拡大縮小されます。
